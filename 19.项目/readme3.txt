gcc编译器

使用gcc指令编译C代码
使用g++指令编译C++代码

编译过程
// 预处理----编译----汇编----链接

// 预处理
// -E 指示编译器对输入文件进行预处理
g++ -E main.cpp -o main.i

// 编译
// -S 告诉g++为C++代码生成汇编语言文件
g++ -S main.i -o main.s

// 汇编
// -c 告诉g++把源代码编译为机器语言的目标代码(二进制语言)
g++ -c main.s -o main.o

// 链接
// -o 将产生的可执行文件用指定的文件名
g++ main.o -o main

执行
./main
